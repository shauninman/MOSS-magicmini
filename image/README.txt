Run ./create.sh from this directory to create MOSS.img

Requires this directory have the sibling directories: 
dtb, kernel, and system containing the dtb files, 
KERNEL, and SYSTEM. 

It also must contain pre.img which can be extracted from
[MagicX_XU_MINI_M_Open.img][0] by adding to this directory
and running ./extract.sh

[0]:https://archive.org/download/magic-x-xu-mini-m-open
